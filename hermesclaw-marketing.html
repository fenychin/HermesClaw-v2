三域架构

每一步都有迹可查，每一次偏差都进入治理流程。

控制内核

Hermes Control Kernel

  • ›意图解析 → TaskEnvelope
  • ›DAG Workflow 生成与编排
  • ›多层记忆管理（会话/项目/组织）
  • ›Agent Policy 治理与自动化等级
  • ›Evaluation + Proposal + Rollback
  • ›AuditLog 写入（治理真相源）
执行运行时

OpenClaw Execution Runtime

  • ›渠道会话与设备在线状态
  • ›连接器动作执行（Connectors/Tools）
  • ›长连接事件流（WebSocket / SSE）
  • ›ExecutionEvent 流回传
  • ›ActionReceipt 存储
  • ›Sandbox 模式与主会话切换
行业插件层

Industry Pack Layer

  • ›行业 Agent 模板（岗位化）
  • ›行业 Workflow Templates
  • ›技能包 + SOP
  • ›行业 KPI Schema + Dashboard
  • ›行业知识包与字段 schema
  • ›行业连接器映射（CRM/ERP/邮件）

# 客户跟进管理与 CRM 同步

## 能力清单 (can_do)

- 自动生成跟进提醒：基于沟通间隔阈值（如 7 天无互动触发提醒）
- 客户阶段智能判断：潜客 → 询盘 → 样品 → 谈判 → 成交 → 复购
- 生成跟进建议话术（基于当前阶段与客户画像）
- 双向同步 CRM：读取客户状态 + 写入跟进记录
- 识别流失风险：沉默超期、竞品切入信号、满意度下降
- 跟进效果追踪：回复率、转化率、阶段推进周期

## 约束条件 (cannot_do)

- 不能自动修改 CRM 中的成交金额与合同条款
- 客户阶段变更须记录变更原因与置信度
- 不得在客户明确要求停止联系后继续生成跟进提醒
- 不能跨项目共享客户沟通记录
- 流失预警判定后须人工确认方可发起挽回流程
- 与 CRM 同步失败时须重试 3 次后告警，不得静默丢弃

## 所需工具 / 连接器

- CRM 读写接口 — 客户状态双向同步
- 邮件/IM 历史读取 — 沟通记录分析
- 客户画像服务 — 阶段判断依据

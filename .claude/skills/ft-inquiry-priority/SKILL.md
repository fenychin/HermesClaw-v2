# 询盘优先级智能评估

## 能力清单 (can_do)

- 多维度优先级评分：客户质量 × 询盘匹配度 × 时效性 × 利润空间
- 关联历史数据：过往成交记录、沟通频次、样品阶段
- 识别高价值信号：具体产品型号、明确交期、合理目标价
- 生成优先级排序看板（高/中/低三档 + 数值分）
- 输出推荐跟进时间窗口与话术方向

## 约束条件 (cannot_do)

- 不能自动将低优先级询盘标记为垃圾或删除
- 评分结果仅供辅助决策，不得替代人工判断
- 不能基于单一负面信号直接降级长期客户询盘
- 历史数据缺失时须明确标注"数据不全，评分仅供参考"
- 不能访问其他 Agent 的私有评分模型

## 所需工具 / 连接器

- CRM 读取接口 — 历史交易与沟通记录
- 海关进出口数据 — 客户采购能力验证
- 客户画像服务 — 客户质量维度评分

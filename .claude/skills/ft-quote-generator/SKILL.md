# 外贸报价生成与优化

## 能力清单 (can_do)

- 基于产品参数、数量、目标市场自动生成报价方案
- 融合实时汇率、海运费/空运费估算、关税预估进行成本核算
- 生成 FOB / CIF / DDP 等多贸易术语报价
- 预设利润率区间（5%/10%/15%/20%）输出建议报价区间
- 报价方案版本管理：每次修改自动递增版本号
- 输出结构化报价明细表（JSON 格式，供前端渲染或 PDF 导出）
- 支持价格有效期设定与自动过期提醒
- 报价单审批流程：草稿 → 审核 → 批准 → 发送

## 约束条件 (cannot_do)

- 不得在未经人工审批的情况下自动发送报价给客户
- 不得低于成本价报价（利润率为负时须明确警示并拒绝生成）
- 不得修改已批准版本的报价数据
- 不得访问企业 ERP 或财务系统中的实际成本数据（仅可使用用户提供的成本参数）
- 汇率波动超过 3% 时须在报价中标注风险提示

## 所需工具 / 连接器

- 汇率数据 API（src/app/api/exchange-rates）
- 产品数据库（Prisma Product 表）
- 报价单 PDF 生成（ft-quotation-pdf 技能）
- 成本核算（ft-cost-accounting 技能）

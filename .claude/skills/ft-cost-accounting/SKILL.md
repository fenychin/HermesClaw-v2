# 产品参数提取与成本核算

## 能力清单 (can_do)

- 从询盘文本/附件中提取产品规格参数（型号、材质、认证要求）
- 关联 ERP/产品库获取最新成本数据
- 实时汇率折算，支持 30+ 币种
- 运费估算：海运/空运/快递/铁路多式联运
- 关税与增值税预估（基于 HS Code + 目的国）
- 生成多贸易术语成本明细：FOB / CIF / DDP
- 利润空间模拟：按预设利润率区间输出建议报价区间

## 约束条件 (cannot_do)

- 汇率以实时 API 为准，不得使用缓存超过 15 分钟的汇率
- 关税税率仅供参考，最终以目的国海关核定为准（须在输出中标注）
- 不能自行调整预设利润率底线
- 数据源缺失时须明确标注，不得用默认值静默填充
- 不得绕过汇率/价格审批流程直接生成正式报价单
- 成本核算结果不得自动提交至客户可见系统

## 所需工具 / 连接器

- ERP 系统接口 — 产品成本与库存
- 实时汇率 API — 多币种换算
- 运费查询 API — 国际物流运费
- HS Code 关税数据库 — 目的国税率查询

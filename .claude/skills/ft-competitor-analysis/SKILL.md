# 竞品分析与市场画像

## 能力清单 (can_do)

- 竞品情报采集：定价、新品发布、促销活动、渠道布局、专利动态
- 海关数据挖掘：进出口量价趋势、主要供应商/采购商格局
- 目标市场画像：市场规模、增长趋势、准入壁垒、渠道结构、文化偏好
- 行业动态摘要生成：政策法规变化、技术迭代、供应链迁移
- 可视化输出：市场份额饼图、价格趋势折线图、SWOT 矩阵
- 可执行建议：优先级市场排序、定价策略参考、渠道切入路径

## 约束条件 (cannot_do)

- 竞品数据采集须遵守目标网站的爬虫协议 (robots.txt)
- 不得采集竞品内部/付费/加密数据
- 分析结论须标注数据来源与时效性（如"基于 2026 Q1 海关数据"）
- 市场预测仅供参考，须附置信区间与假设前提
- 不得将客户专属市场分析报告共享给其竞品
- 涉及政策解读的内容须标注"非法律建议"

## 所需工具 / 连接器

- 海关进出口数据 — 贸易流量分析
- 行业数据库 — 市场规模与趋势
- 汇率 API — 价格对比换算
- 新闻聚合 API — 竞品动态监控
- 数据可视化引擎 — 图表生成

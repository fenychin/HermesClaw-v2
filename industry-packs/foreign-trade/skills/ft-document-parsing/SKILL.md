# 单证解析与合规检查

## 能力清单 (can_do)

- OCR 解析扫描件/PDF：提单、发票、装箱单、产地证、保险单
- 关键字段提取：发货人/收货人、船名航次、集装箱号、HS Code、毛净重
- 跨单据字段一致性校验：数量、金额、唛头、HS Code 一致性
- 信用证条款匹配：自动比对 L/C 条款与单据内容，识别不符点
- 常见错误识别：拼写错误、日期逻辑矛盾、签章缺失、背书不完整
- 输出差异报告与修正建议（按严重程度分级）

## 约束条件 (cannot_do)

- 不得直接修改原始单据内容
- 不符点判定结论仅供参考，最终以银行/海关审核为准
- OCR 置信度 < 0.85 的字段须标记为"需人工核验"
- 不得将单据内容用于模型训练或跨项目共享
- 信用证审单结果必须由持证单证员复核
- 不能替代信用证修改申请流程

## 所需工具 / 连接器

- OCR 文档解析引擎 — 扫描件文字提取
- 海关进出口数据 — HS Code 校验
- 信用证管理接口 — L/C 条款比对
- 提单查询 API — 船期与状态验证

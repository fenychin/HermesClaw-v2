# 多币种报价单 PDF 生成

## 能力清单 (can_do)

- 生成专业格式报价单 PDF（A4 标准排版）
- 多币种双栏显示（客户币种 + 人民币对照）
- 自动填充：产品明细表、单价、总价、贸易术语、交期
- 嵌入公司抬头、Logo、电子签章、条款页脚
- 版本管理：每次修改自动递增版本号（v1 → v2 → …）
- 在线审批流：草稿 → 审核 → 批准 → 发送
- 历史版本对比与回滚

## 约束条件 (cannot_do)

- 报价单 PDF 生成后必须进入审批流程，不得直接发送客户
- 不得修改已批准的历史版本报价单
- 电子签章使用须通过权限校验
- 报价有效期默认 15 天，超期须重新核算
- 不得在未授权情况下使用客户 Logo 或品牌信息
- PDF 生成出现格式错误时须阻断发送并告警

## 所需工具 / 连接器

- ERP 系统接口 — 产品与价格数据
- 实时汇率 API — 多币种换算
- PDF 渲染引擎 — 报价单生成
- 电子签章服务 — 签署授权
- 审批工作流引擎 — 流程管理

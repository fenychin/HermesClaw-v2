# 一、项目背景

HermesClaw-v2 是一个面向中小企业（小B场景）的 AI 数字员工基础平台。

系统架构定义如下：
- Hermes：云端规划与记忆控制面，负责规划、编排、记忆、治理、动态 Harness 演化
- OpenClaw：用户移动端执行与连接器数据面，负责执行任务、采集数据、触发连接器动作、回传执行日志

该平台支持：
- 多智能体协同
- 项目空间化工作
- 文件与知识沉淀
- 技能与连接器复用
- 行业 Workflow 自动生成
- 动态 Harness 自我进化与升级

首个优先切入行业：
- 外贸

Web UI 是核心入口，移动端是执行与连接辅助入口。
